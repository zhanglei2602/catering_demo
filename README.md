# 私房菜 AI 菜单 Demo

一个从零搭建的演示项目：

- 前端使用原生 HTML + CSS + JavaScript 展示私房菜菜单体验
- 后端使用 Python 提供菜单生成逻辑和接口
- 菜单会根据人数、用餐场景、预算、口味偏好、忌口和备注生成一套模拟私房菜席面

## 运行方式

```bash
python app.py
```

启动后访问：

```text
http://127.0.0.1:8000
```

后厨调度页：

```text
http://127.0.0.1:8000/kitchen
```

## 目前包含的能力

- 模拟多种私房菜菜品数据，包括前菜、汤品、海鲜、禽类、肉菜、时蔬、主食、甜品
- 根据预算和人数动态控制菜量与价格区间
- 根据偏好、忌口和场景做筛选与打分
- 为每道菜生成“为什么推荐”和“上桌表达”
- 输出完整菜单标题、菜单故事、主厨备注，适合做 demo 展示
- 提供后厨实时调度页，厨房持续运转，可手动加入新桌并观察动态出餐过程

## 后续可扩展方向
